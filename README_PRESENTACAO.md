# Apresentação — Simulador de Investimentos

Este README contém instruções rápidas para usar a apresentação e publicar o projeto no GitHub.

## Abrir slides localmente

1. No Windows (PowerShell):

```powershell
cd "C:\Users\gabri\Área de Trabalho\simulador-investimento"
start PRESENTACAO_SLIDES.html
```

2. No navegador: navegar com as setas ou botões; imprimir → Salvar como PDF.

## Sugestão: substituir imagens por screenshots

- Suba screenshots do app para a pasta `images/` com nomes: `screenshot_simulador.png`, `screenshot_login.png`.
- Atualize `PRESENTACAO_SLIDES.html` para apontar as imagens nos slides relevantes.
